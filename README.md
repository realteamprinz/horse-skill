## What is horse.skill?

A species-specific extension of paw.skill that adds deep equine intelligence — riding partnership memory, ground manners tracking, herd dynamics mapping, health and lameness monitoring, and temperament profiling across disciplines.

paw.skill knows your pet. horse.skill knows your HORSE.

## Install

Requires [paw.skill](https://github.com/realteamprinz/paw-skill) as parent.

```bash
# OpenClaw
clawhub install realteamprinz/paw
clawhub install realteamprinz/horse

# Hermes
hermes skills install paw-skill
hermes skills install horse-skill

# Claude Code
claude install-skill https://github.com/realteamprinz/paw-skill
claude install-skill https://github.com/realteamprinz/horse-skill
```
